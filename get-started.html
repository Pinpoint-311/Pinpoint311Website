Deploy Pinpoint 311 in 3 steps

1
Clone the repository
You'll need Docker and Docker Compose installed on your server or VM.
git clone https://github.com/Pinpoint-311/Pinpoint-311.git
cd Pinpoint-311
2
Configure your environment
Copy the example env file and set your database password and secret key.
cp .env.example .env
# Edit .env — set DB_PASSWORD and SECRET_KEY
3
Pull images and launch
Uses prebuilt images — no local build required. Your portal will be live in minutes.
docker compose -f docker-compose.yml \
-f docker-compose.prod.yml pull
docker compose -f docker-compose.yml \
-f docker-compose.prod.yml up -d
